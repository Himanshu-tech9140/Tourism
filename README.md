# 🌍 North India Tourism Platform

A modern full-stack tourism web application focused on promoting North India travel destinations with advanced features like multi-language support, AI chatbot, and real-time payment integration.

---

## 🚀 Project Overview

This project is built using **HTML, CSS, JavaScript, JSP, and JDBC** with backend database integration. It provides users a smart and interactive travel experience with booking and assistance features.

---

## 🛠️ Tech Stack

**Frontend:**
- HTML5
- CSS3
- JavaScript

**Backend:**
- JSP (Java Server Pages)
- JDBC
- Servlets (if used)

**Database:**
- MySQL

**Payment Gateway:**
- Razorpay (Real-time payment integration)

---

## ✨ Key Features

- 🏔️ Explore North India tourist destinations  
- 💳 Real-time payment integration using Razorpay  
- 🌐 Multi-language support for better accessibility  
- 🤖 AI Chatbot for user assistance and travel guidance  
- ⏰ 24×7 service availability  
- 📄 Dynamic content using JSP  
- 🗄️ Database-driven information system using JDBC  
- 📱 Responsive and user-friendly UI
